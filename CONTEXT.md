# CONTEXT — CustomArc ubiquitous language

Glossary only. No implementation details.

| Term | Meaning |
|---|---|
| **CustomArc** | The platform — "arc for customization". Users customize 3D-previewed products in the browser and order the physical item. |
| **Blank** | A base physical product offered for customization (Phase 1: mug, phone case). A blank is data + assets (3D model, print template), not code. |
| **Customizer** | The in-browser 3D editor where a user applies artwork to a blank — manually or via AI. |
| **Design** | A user's saved customization of a blank: artwork, text, placement. Owned by the user, stored as a document. |
| **Generation** | One AI text→texture invocation that produces artwork for a design. Consumes credits. |
| **Credits** | Prepaid units consumed by generations. Some granted free; more purchasable. Balance derives from an append-only ledger. |
| **Print File** | The manufacturer-agnostic artifact produced from a design: flat print-resolution artwork + placement spec. Consumed by the print partner today, by CustomArc's own machines in a later phase. |
| **Print Partner** | Third party that manufactures and ships orders in Phase 1. |
| **Geometry Product** | Future product whose 3D shape itself is AI-modified (e.g. cat-shaped mug). Preview and fulfillment are later phases. |
| **Creator** | Future marketplace seller who publishes designs on CustomArc for commission (later phase). |
| **Bulk Lead** | A B2B inquiry captured via the "bulk order? contact us" form in Phase 1. The full B2B suite comes in a later phase. |
